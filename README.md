# Okultis Web

Corporate website for [Okultis](https://okultis.com) — a digital studio specializing in software development, UX/UI design, digital transformation, and AI.

## Stack

- HTML, CSS, vanilla JavaScript
- Space Grotesk font (Google Fonts)
- reCAPTCHA v3 for contact form spam protection
- Google Analytics

## Structure

```
├── index.html            # Main landing page
├── contact.html          # Contact page with form & map
├── privacy-policy.html   # Privacy policy
├── terms.html            # Terms of service
├── css/
│   └── style.css         # All styles
├── js/
│   ├── main.js           # Navigation, animations, form handling
│   └── cookies.js        # Cookie consent logic
├── assets/
│   └── images/           # Logos & client images
└── send.php              # Server-side form handler
```

## Development

No build step required. Open `index.html` in a browser or serve with any static file server.
